# umineco.org

[umineco.org](https://umineco.org) のリポジトリです

## Build

`/_site` ディレクトリ以下にビルドの成果物が生成されます。

```
npm install
npm run build
```

## Development

`/src` ディレクトリ以下の更新を監視して、自動的にビルドを実行します。

```
npm install
npm run start
```

## Deployment

`main` ブランチにマージすると、 Cloudflare Pages によって自動的にデプロイされます。
